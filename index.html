Porque "Ursa"?

A Ursa foi criada para ser usada por qualquer pessoa, desde que tenha conhecimento no discord ou não.
Ela tem um prefixo muito fácil de usar, layout de comando chamativo e um menu de ajuda de fácil entendimento e leitura.
Os comandos da Ursa são divertidos, então você nunca terá um momento chato no discord com seus amigos!

Qual o prefixo?

Fácil! Mencionando ela você poderá receber uma mensagem rápida monstrando o prefixo e o comando de ajuda!

Ela é Open Source?

Felizmente ainda não :( Pois como a diversos repositório em conjunto alem das Api Grabber, Faze-la open source seria uma tarefa difícil.
